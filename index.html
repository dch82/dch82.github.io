About this website

This website uses classless Pico CSS for styling and Iosevka variants for the font. Most text is done in Iosevka Aile, a variable-width variation of Iosevka that I find easier to read. Any fixed-width text is rendered in regular Iosevka. Apart from these imports, the website is done in handwritten HTML and some CSS. Everything on this website is licensed as CC-by-SA.
